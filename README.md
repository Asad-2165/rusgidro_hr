# HR-портал «РусГидро» — Система мотивации персонала

## Описание проекта

Комплексная цифровая платформа для повышения вовлеченности, развития карьеры и формирования культуры признания среди 30 000+ сотрудников ПАО «РусГидро».

### 17 модулей мотивации:
1. **Энергия благодарности** — публичные благодарности коллегам
2. **Карьерный навигатор** — персональные карьерные треки
3. **Марафон безопасности** — игровые челленджи по безопасности
4. **Рационализаторские предложения** — автоматический расчет экономического эффекта
5. **Клуб наставников** — накопительная мотивация за адаптацию
6. **Семейные мероприятия** — экскурсии, конкурсы, старты
7. **Забота о себе** — wellness и анти-выгорание
8. **Герой смены** — еженедельное полевое голосование
9. **Кросс-дивизиональный обмен** — мастер-классы между филиалами
10. **Эко-челлендж** — экологические соревнования
11. **Долгосрочная лояльность** — градация поощрений по стажу
12. **Тренажер ценностей** — геймифицированные квесты
13. **Приведи коллегу** — реферальная программа
14. **Инженерные конкурсы** — ежемесячные соревнования
15. **Персональный дашборд** — визуализация прогресса
16. **Мотивация в оптимизации** — методики в бизнес-процессах
17. **Управление мотивацией** — система управления заинтересованностью

## Технологический стек

### Frontend
- HTML5, CSS3, JavaScript (vanilla)
- Адаптивный дизайн (mobile-first)
- CSS Grid, Flexbox
- Intersection Observer API для анимаций

### Backend
- **FastAPI** — современный асинхронный веб-фреймворк
- **Pydantic** — валидация данных
- **SQLAlchemy** — ORM для работы с БД
- **PostgreSQL** — основная база данных
- **Redis** — кэширование и сессии
- **JWT** — аутентификация

### Инфраструктура
- Docker + Docker Compose
- Yandex Cloud / VK Cloud
- CI/CD через GitHub Actions
- Nginx — reverse proxy

## Установка и запуск

### Локальный запуск (разработка)

```bash
# Клонирование репозитория
git clone https://github.com/your-org/rusgidro-hr-portal.git
cd rusgidro-hr-portal

# Создание виртуального окружения
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows

# Установка зависимостей
pip install -r requirements.txt

# Запуск backend
uvicorn api.main:app --reload --host 0.0.0.0 --port 8000

# Frontend — открыть index.html в браузере
# или запустить через Live Server
```

### Docker запуск

```bash
# Сборка и запуск
docker-compose up --build

# Backend будет доступен на http://localhost:8000
# Frontend на http://localhost:80
# API Docs: http://localhost:8000/docs
```

| GET | `/api/project/metrics` | Метрики проекта |
| GET | `/api/modules` | Список модулей |
